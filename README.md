# Fun Olympics API

Fun Olympics API is created using Nest JS, MySQL and TypeORM. With this API, users can easily interact with the website's games, live broadcasts, videos, news, and comments section.

Major Tools used:

- Nest JS
- MySQL
- TypeORM
- Nodemailer
- Passport JS
- Bcrypt JS
- Class Transformer
- Class Validator
- TypeORM Paginate
